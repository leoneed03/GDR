# GDR: global depth reconstructor

Global reconstruction from RGB-D data

make shure you have dependencies preinstalled:

## Dependencies

1. nvcc (Cuda Development Kit)
2. GLEW
3. GLUT
4. Eigen
5. PCL
6. OpenCV
7. Boost
8. TBB
9. GTest
10. GTSAM


## Project

```
mkdir build
cd build
cmake ..
make
ctest
```
