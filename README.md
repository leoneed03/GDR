# GDR: global depth reconstrutor

Global reconstruction from RGB-D data

make shure you have dependencies preinstalled:

## Dependencies

1. nvcc (Cuda Development Kit)
2. GLEW
3. GLUT
4. Eigen
5. PCL
6. OpenCV
7. Boost
8. TBB
9. GTSAM

## SiftGPU
To build SiftGPU:

```
cd siftgpu
make 
```

## Project

```
mkdir build
cd build
cmake ..
make
```
